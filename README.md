<h1 align="center"> Hexlet Project "Brain Games".</h1>

## Install

Clone this repository
```
$ git clone
```
Type npm link in your terminal
```
$ npm link
```
Type make install in your terminal
```
$ make install
```
### Presentation of Brain-even
